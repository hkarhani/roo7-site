About this chart

  • Platform performance aggregates ROO7 managed trading accounts and strategies, normalized each hour.
  • Benchmark selector toggles between the composite basket or individual BTC/ETH/SOL/BNB/XRP spot references.
  • Benchmark prices are sourced from Binance spot klines (1-hour intervals) and stored internally.
  • Each point shows cumulative change from the start of the selected window; the first value is always 0% by design.
  • Data updates shortly after each hour. Times shown in UTC.
